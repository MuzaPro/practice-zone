The Flexbox Layout Model

A modern CSS layout solution

What is Flexbox?

Flexbox (Flexible Box Module) is a one-dimensional layout model designed to provide a more efficient way to arrange, align, and distribute space among items in a container, even when their size is unknown or dynamic.

Before Flexbox, we had to rely on floats, positioning, and table properties which were often limiting and cumbersome. Flexbox makes complex layouts simple!

Key Concepts

Flex Container

The parent element that has display: flex applied to it.

Flex Items

The direct children of the flex container.

Main Axis

The primary axis along which flex items are placed (horizontal by default).

Cross Axis

The axis perpendicular to the main axis (vertical by default).

Creating a Flex Container

To create a flex container, set the display property to flex or inline-flex:

.container {
    display: flex;
}
