What is Tailwind CSS?

Tailwind CSS is a utility-first CSS framework that lets you build designs directly in your markup. Instead of pre-designed components like other CSS frameworks, Tailwind provides low-level utility classes that let you build completely custom designs without ever leaving your HTML.

With Tailwind, you style elements by applying pre-existing classes directly in your HTML. This approach allows for rapid development while maintaining complete design flexibility.

Traditional CSS Approach

.card {
  background-color: white;
  border-radius: 0.5rem;
  padding: 1.5rem;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

.card-title {
  font-size: 1.25rem;
  font-weight: 600;
  color: #2563eb;
  margin-bottom: 0.75rem;
}

Traditional CSS requires you to create custom classes and switch between HTML and CSS files.

Tailwind Approach

<div class="bg-white rounded-lg p-6 shadow-md">
  <h2 class="text-xl font-semibold text-blue-600 mb-3">
    Card Title
  </h2>
</div>

With Tailwind, you apply utility classes directly in your HTML markup.

Why Use Tailwind CSS?

Advantages

  • Faster development - Build interfaces quickly without writing custom CSS
  • Responsive design - Built-in responsive modifiers like md: and lg:
  • Customizable - Easily extend the default configuration
  • No naming fatigue - No need to invent class names

Considerations

  • Verbose HTML - Markup can become long with many utility classes
  • Learning curve - Need to learn Tailwind's utility classes
  • Build step required - For production, you need a build process

When to Use Tailwind CSS

  • New Projects

    Tailwind is ideal for new projects where you want to build a custom UI quickly without accumulating CSS technical debt.

  • Prototyping

    The utility-first approach makes it perfect for quickly prototyping ideas without writing custom CSS.

  • Design Systems

    Tailwind's configuration system makes it excellent for implementing and maintaining design systems.

Flexbox with Tailwind

Tailwind makes working with Flexbox incredibly simple by providing utility classes for all common Flexbox properties:

Flexbox with Tailwind:

Item 1
Item 2
Item 3
<div class="flex justify-between items-center">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

Common Flexbox Utilities:

  • flex - Create a flex container
  • flex-row / flex-col - Direction
  • justify-start / justify-center / justify-between - Main axis
  • items-center / items-start / items-end - Cross axis
  • flex-wrap / flex-nowrap - Wrapping
  • gap-{size} - Gap between items

Dark Mode with Tailwind

Tailwind makes it simple to implement dark mode with the dark: variant prefix. Simply add dark: before any utility class to have it applied only in dark mode.

Dark Mode Examples:

Code Example:

  </h4>

Getting Started with Tailwind

There are several ways to set up Tailwind CSS in your project:

Method 1: CDN (Quick Start)

Add the Tailwind CDN to your HTML file (not recommended for production):

<script src="https://cdn.tailwindcss.com"></script>

Method 2: Install via NPM (Recommended)

1. Install Tailwind and its dependencies:

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

2. Configure your tailwind.config.js file:

module.exports = {
  content: ["./src/**/*.{html,js}"],
  theme: {
    extend: {},
  },
  plugins: [],
}

3. Add the Tailwind directives to your CSS:

@tailwind base;
@tailwind components;
@tailwind utilities;

Method 3: Framework Integrations

Tailwind works seamlessly with popular frameworks:

  • Next.js: npm install -D tailwindcss
  • React: Works with Create React App
  • Vue: Compatible with Vue CLI
  • Angular: Can be integrated with Angular projects
